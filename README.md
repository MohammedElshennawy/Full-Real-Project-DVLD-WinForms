# 🚦 DVLD - Driving & Vehicle License Department System

A full desktop application developed using **C#, Windows Forms, ADO.NET, SQL Server, and 3-Tier Architecture** to simulate a real-world Driving & Vehicle License Department (DVLD).

The system manages driving license services, applicants, drivers, tests, users, applications, license issuance, renewals, detentions, international licenses, and other related operations while applying real business rules and workflows.

---

## 🚀 Features

✔️ People Management  
✔️ User & Permissions Management  
✔️ Driver Management  
✔️ License Classes Management  
✔️ Application Types Management  
✔️ Test Types Management  
✔️ Vision, Written, and Practical Driving Tests  
✔️ New Driving License Issuance  
✔️ License Renewal  
✔️ Lost License Replacement  
✔️ Damaged License Replacement  
✔️ License Detain & Release  
✔️ International Driving License Issuance  
✔️ License History Tracking  
✔️ Authentication & Authorization System  
✔️ Application Workflow Management
